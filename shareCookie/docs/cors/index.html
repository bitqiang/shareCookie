# cors使用说明

# 功能

  1. 将不支持跨域的使其支持跨域

# 原理

  1. 修改后端返回响应头以支持跨域Access-Control-Allow-Origin: * (后端返回如果有改响应头则修改为*,没有则添加该条响应头)

# 注意的点

  1. 启动浏览器时，会自动读取上一次配置，进行跨域处理
  2. 当删除规则时，之前支持跨域的依旧支持跨域（由于是使用chrome插件的监听功能， 暂未找到可以取消监听的方法，如需立即生效，可能需要重启浏览器）
  3. 添加规则时会立即生效
